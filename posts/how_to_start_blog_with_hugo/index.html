如何用hugo搭建个人博客

一，下载hugo

  1. 下载地址

  2. 找到 hugo_xxx_Windows-64bit.zip 下载

  3. 解压,把 hugo.exe 所在的目录加到PATH中

  4. 重启终端，运行 hugo version 查看版本

二，Create a New Site(新建一个"Hugo site”)

    hugo new site 你的github用户名.github.io-generator

上面这个操作会在当前目录新建一个名为“你的github用户名.github.io-generator”的文件夹，并在“你的github用户名.github.io-generator”文件夹中新建一个"Hugo site”。

三，Add a Theme(添加主题)

    cd 你的github用户名.github.io-generator
    
    git init
    
    git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke

    echo 'theme = "ananke"' >> config.toml

上面这四个操作分别会进入刚才新建的文件夹、创建一个名为 .git 的子目录、下载ananke主题、配置主题为ananke。

四，Add Some Content(写博客内容)

    hugo new posts/my-first-post.md

上面这条命令会新建md博客文件，博客文件的位置是”./content/posts”。

打开博客文件，然后就可以写博客了。

draft(草稿)默认是true，写完博客后要改成false。

五，Start the Hugo server(启动Hugo服务器)

    hugo server -D

复制网址，就可以在浏览器中打开了。如果按了“ctrl+c”将会关闭服务器。
